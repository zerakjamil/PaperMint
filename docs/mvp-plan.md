# PaperMint MVP Plan

## Implemented Vertical Slice

1. Structured exam project model with typed blocks
2. Three-panel editor with metadata, print preview, properties
3. Question insertion, reorder, duplicate, delete
4. Local save/open with JSON validation
5. PDF and DOCX export from the structured model

## Remaining Enhancements

1. Native Tauri filesystem and dialog plugin integration
2. More precise pagination algorithm for preview and exports
3. Autosave and recent project list
4. Extended block-level validation and keyboard shortcuts
